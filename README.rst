QML: A Python Toolkit for Quantum Machine Learning
==================================================

.. attention::

	Further development of the QML package is now in the https://github.com/qmlcode/qmllib repository. The pip-package ``qmllib`` is a streamlined version of QML's Fortran core functionality with fewer dependencies. We'd like to encourage you to direct any future issues or pull requests to this new repository. **This repository is now archived**.

	Both the ``master`` and ``develop`` branches have been left for the archive.
